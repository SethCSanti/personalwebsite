# Personal Portfolio Website

## Overview

This repository contains the source code for my personal website. The site functions as both a professional portfolio for my computer science work and a place to present my writing as an author.

The site is divided into two primary sections:

1. Computer Science Portfolio
2. Author & Writing

The goal of this project is to create a clean, professional hub where visitors can explore my technical work while also seeing the creative projects I produce outside of programming.

---

# Site Structure

The website is organized into two main sections.

---

# 1. Computer Science Portfolio

This section highlights my work as a computer science student and developer.

At the top of the page there will be a downloadable resume button so visitors can quickly access my resume.

This section focuses on software development, programming projects, and technical experimentation.

## Features

- Resume download button at the top of the page
- Project showcase with descriptions
- Links to GitHub repositories
- Technologies used for each project
- Screenshots or demos where applicable
